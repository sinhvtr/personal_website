# Dr. Vu Trong Sinh - Personal Portfolio

Welcome to the source code of my personal academic and professional portfolio website. 

This is a modern, responsive, and bilingual landing page designed to showcase my academic background, teaching experience, research publications, and professional certifications.

## 🌟 Key Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Dark/Light Mode**: User preference is automatically saved via `localStorage` for a seamless experience.
- **Bilingual Support (EN/VI)**: Built-in internationalization allowing users to switch between English and Vietnamese seamlessly.
- **Scroll Animations**: Smooth reveal animations as the user scrolls down the page.
- **Modern UI/UX**: Clean, academic-focused aesthetic using CSS variables for easy theme customization.

## 🛠️ Technologies Used

This project is built using foundational web technologies without the need for heavy frameworks or build tools, making it extremely fast and easy to host on static hosting platforms like GitHub Pages.

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, Flexbox/Grid layouts, and CSS variables.
- **Vanilla JavaScript**: DOM manipulation, event handling, theme toggling, and multi-language support.
- **FontAwesome**: Icons for UI elements.

## 📂 Project Structure

```
.
├── index.html       # The main HTML document containing the structure and content
├── style.css        # All CSS styles, theme variables, and responsive media queries
├── script.js        # JavaScript logic (theme toggle, i18n, scroll animations)
└── README.md        # Project documentation
```

## 🚀 How to Run Locally

Since this is a static website, running it locally is incredibly simple:

1. Clone this repository:
   ```bash
   git clone https://github.com/sinhvtr/sinhvtr.github.io.git
   ```
2. Open the directory and simply double-click on `index.html` to view it in your browser.
3. Alternatively, you can use an extension like **Live Server** in VS Code for live reloading during development.
